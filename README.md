Шаг 1. Выбор справочников

Мои справочники:
1. Отделы
2. Сотрудники

Поля справочника «Отделы»:
- id
- название
- описание
- дата_создания
- этаж
- премиальный_коэффициент

Поля справочника «Сотрудники»:
- id
- фамилия
- имя
- отчество
- должность
- дата_приёма
- табельный_номер
- оклад
- department_id

Справочник «Сотрудники» зависит от справочника «Отделы»: каждый сотрудник относится к определённому отделу.
department_id в справочнике «Сотрудники» - значения из справочника «Отделы».

Типы данных:
- текст: название, описание, фамилия, имя, отчество, должность;
- дата: дата_создания, дата_приёма;
- целые числа: этаж, табельный_номер;
- числа с фиксированной запятой: премиальный_коэффициент, оклад.
